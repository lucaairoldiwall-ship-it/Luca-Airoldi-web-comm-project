#  The M.A.L.L.S.

> A Punk Rock Band Website\
> By **Luca Airoldi**

------------------------------------------------------------------------

##  Overview

**The M.A.L.L.S.** is a web project designed to promote a punk rock band
online.

The website allows users to:  
-  Discover the band and its members
-  Explore upcoming live shows
-  Interact with the band

------------------------------------------------------------------------

##  Project Idea

The goal of the project is to create a simple but engaging platform that
helps a band: 
- Gain visibility
- Present their identity
- Connect with fans

------------------------------------------------------------------------

##  Tech Stack

-   **Static Site Generator:** GetNikola
-   **Frontend:** HTML, CSS, JavaScript
-   **Other Features:** Cookies, Mailto form

------------------------------------------------------------------------

##  Design

-    Dark theme
-    Purple accents
-    Punk rock inspired aesthetic

------------------------------------------------------------------------

##  Members Page

-   Band presentation
-   Images of members
-   Roles and descriptions

------------------------------------------------------------------------

##  Live Page

Displays a table of upcoming concerts: 
-  Date
-  Venue
-  City

------------------------------------------------------------------------

##  Cookies

Used to store user preferences: 
- Name
- Favorite section

------------------------------------------------------------------------

##  Contact Form

-   Collects user input
-   Generates an email using `mailto`

------------------------------------------------------------------------

##  Features

-    Simple and clean structure
-    Interactive elements with JavaScript
-    User personalization via cookies
-    Easy to deploy (static site)
